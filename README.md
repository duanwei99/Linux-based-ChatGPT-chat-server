# Linux-based-ChatGPT-chat-server
基于Linux的ChatGPT聊天服务器



使用注意事项：

因为设计到C++和python代码的信息传输，所以很多中间文件写的是绝对路径，需要根据自己的文件所在调整相关路径

mysql默认的账户密码在main.cpp中，需要调整服务器端对应的配置

需要在gpt.py文件中添加自己的gpt-key
